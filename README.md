# danmu

> A Vue.js project

## Build Setup

``` bash
# 前台安装依赖，根目录下执行：
npm install

# 前台项目运行,根目录下执行：
npm run dev

# 后台安装依赖，server目录下：
npm install

# 后台项目运行，server目录下执行：
npm run build

# 数据库连接
// 数据库连接方式见下

# 浏览器访问：localhost:5555
```

For detailed explanation on how things work, checkout the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
